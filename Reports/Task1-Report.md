# Task 1 Report

## Aim
To create a simple smart contract for transferring tokens and storing user balances.

## Technologies Used
- Solidity
- Hardhat
- Node.js

## Steps Performed
1. Initialized a Hardhat project.
2. Wrote `SimpleTokenTransfer.sol` with beginner-friendly comments.
3. Created a `mapping` to store balances and an `event` for transfers.
4. Created a deployment script.
5. Deployed the contract to a local Hardhat test network.

## Result
The smart contract compiled successfully. I was able to deploy it locally and verify the initial balance of the owner account.

## Conclusion
This task was a great introduction to Solidity basics. I learned how state variables work on the blockchain and how to securely deduct from a sender and add to a receiver.
