# CodeTech Tasks

This repository contains the completion of my 4 blockchain internship tasks for CodeTech. 
The goal of these tasks is to demonstrate fundamental concepts of blockchain development, including smart contracts, ERC20 tokens, decentralized finance (DeFi), and private blockchains.

## Project Structure

- **Task-1-Token-Transfer-System**: A simple smart contract to simulate user balances and transferring tokens.
- **Task-2-ERC20-Token**: A custom ERC20 token named CodeTech Token (CTT) using OpenZeppelin.
- **Task-3-DeFi-Lending-DApp**: A basic decentralized application for depositing, borrowing, and repaying tokens using React and a smart contract backend.
- **Task-4-Private-Blockchain**: A private blockchain setup using Ganache, featuring a simple on-chain message board application.
